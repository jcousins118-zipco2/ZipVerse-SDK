# Contributing

This repository is currently focused on the public ZipVerse contract surface.

Useful contributions include:

- corrections and clarity improvements to public documentation;
- safe integration examples;
- schema validation feedback;
- mock-runtime design proposals that do not depend on private implementation details.

Please keep contributions within the public boundary described in [`docs/integration-boundary.md`](docs/integration-boundary.md).

Do not submit:

- private ZipVerse source code or internal documents;
- secrets, credentials, customer data, or operational configuration;
- reverse-engineered private runtime details;
- claims that draft version `0.1` schemas are production-stable.

Before a public software licence is selected, contributions should be limited to discussion and review unless the repository owner explicitly approves otherwise.
